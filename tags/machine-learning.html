Meta Learners

December 13, 2025 · 2 min read
Hinny Tsang
Data Scientist @ Pollock Asset Management

A paradigm that uses supervised machine learning models to estimate treatment effects.

Questions before starting1^11​

  1. What is the treatment?
  2. What is the outcome?
  3. What are the cofounders (variables that affect both treatment and outcome)?

Terminologies​

  • ATE: Average Treatment Effect

  • ATT: Average Treatment effect on the Treated

  • ATU: Average Treatment effect on the Untreated

  • CATE: Conditional Average Treatment Effect

  • ITE: Individual Treatment Effect (sometimes also called PTE: Personal Treatment Effect)

  • HTE: Heterogeneous Treatment Effect (another name for CATE)

  • Y(T)Y(T)Y(T): Potential outcome

  • TTT: Treatment assignment indicator (T=1T=1T=1 if treated, T=0T=0T=0 if untreated)

  • Feature:

    • XXX: Observed covariates (Controls)
    • WWW: Cofounders that affect both treatment and outcome (Controls and variables)
    • ZZZ: Other covariates that affect only treatment (Instruments)

Assumptions​

  1. Conditional independent asssumption (CIA, aka unconfoundedness), assumes that given a set of observed covariates XXX, the treatment assignment is independent of the potential outcomes. Formally, T⊥(Y(0),Y(1))∣XT \perp (Y(0), Y(1)) \mid XT⊥(Y(0),Y(1))∣X.

Formulation​

  • The CATE can be estimate by

    τ(t0,t1,x)=E[Y(t1)−Y(t0)∣X=x]\tau(t_0, t_1, x) = E[Y(t_1) - Y(t_0) \mid X=x] τ(t0​,t1​,x)=E[Y(t1​)−Y(t0​)∣X=x]

Meta-Learners​

Below are some common methods.

S-learner (Single learner)​

Using single supervised learning model to estimate the outcome YYY given both treatment TTT and covariates XXX as features.

i.e.

τ=μ(X,T=1)−μ(X,T=0)\tau = \mu(X, T=1) - \mu(X, T=0) τ=μ(X,T=1)−μ(X,T=0)

Where μ(X,T)\mu(X, T)μ(X,T) is the predicted outcome from the supervised learning model using both XXX and TTT as features.

T-learner (Two learners)​

Using two separate supervised learning models to estimate the outcome YYY for treated and untreated groups respectively.

i.e.

τ=μ1(X)−μ0(X)\tau = \mu_1(X) - \mu_0(X) τ=μ1​(X)−μ0​(X)

Where μ1(X)\mu_1(X)μ1​(X) is model outcome trained on the treated group (T=1T=1T=1), and μ0(X)\mu_0(X)μ0​(X) is the outcome from the model trained on the untreated group (T=0T=0T=0).
